# Biau Labs 展示系统

基于 React、Vite、TypeScript 和 Semi Design 构建的动态展示网站，用于组织 AI 应用、全栈开发、游戏项目、案例中心和博客内容。

## 本地开发

```bash
npm run dev
npm run build
npm run lint
npm run preview
```

## 技术栈

- React 19
- Vite
- TypeScript
- Semi Design

## 部署

推荐先部署到 Cloudflare Pages。详细方案见 [docs/deployment.md](docs/deployment.md)。
